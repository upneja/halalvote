Key Flows

Submit Topic

1. User writes title + description

2. Submit вҶ’ API route

3. Claude API generates 2-4 tags

4. Topic + tags saved to DB

5. Appears in feed immediately

Vote + Comment

1. User sees topic (no results yet)

2. Taps Halal or Haram

3. Vote saved, % results revealed

4. Comment section unlocks

5. Can add comment + optional scholar cite
